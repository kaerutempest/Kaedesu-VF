⚠️ Gagal Memuat Aplikasi (Deployment Error)

Sistem mendeteksi halaman web Anda masih blank putih. Ini terjadi karena Anda mengunggah kode sumber mentah (.tsx) langsung ke GitHub Pages tanpa melakukan proses kompilasi (build) terlebih dahulu. Browser tidak bisa membaca file .tsx secara langsung.

Cara Memperbaiki (Dua Solusi Tercepat):
  1. Solusi Tercepat & Termudah: Hubungkan ke Vercel
    Buka vercel.com, hubungkan repository GitHub Anda, lalu pilih framework Vite. Vercel akan otomatis meng-compile dan meng-host web Anda setiap kali push!
  2. Gunakan GitHub Pages secara manual (dengan Build):
    Jalankan perintah npm run build di komputer Anda untuk menghasilkan folder dist. Kemudian upload hanya isi folder dist tersebut ke repository GitHub Pages Anda.
